### Built With

- [Golang](https://go.dev/)

### Prerequisites

* golang
* docker

### Installation

1. Clone the repo
   ```sh
   git clone git@github.com:arkavidia-hmif/itfest-backend-2.0.git
   ```
2. Run shell command
   ```sh
    go mod tidy
   ```
3. Run using go or using docker
   ```js
    go run main.go
    // or
    docker-compose up -d
   ```
